# ROS Basics in 5 Days (C++)

This course is a fast, hands-on path into ROS for someone who can already program but hasn't worked in robotics middleware before. It walks from the shape of a ROS system (nodes, the graph, workspaces) through the three core communication patterns — topics, services, and actions — building real C++ nodes at every step, then closes with the debugging tools and local setup you'll rely on for every ROS project afterward. By the end you should be able to read an unfamiliar ROS package, understand its interface, and write your own nodes that publish, subscribe, serve, and act correctly.

1. [Introduction](01-introduction.md) — Unit for previewing the contents of the course.
2. [ROS Deconstruction](02-ros-deconstruction.md) — A guide to correctly following this course.
3. [ROS Basics](03-ros-basics.md) — Basic ROS concepts: nodes, parameter server, environment variables, roscore, etc.
4. [Understanding ROS Topics - Publishers](04-understanding-ros-topics-publishers.md) — What ROS topics, publishers, and topic messages are, and how they work.
5. [Understanding ROS Topics - Subscribers & Messages](05-understanding-ros-topics-subscribers-and-messages.md) — What a subscriber is, how to create one, and how to create your own message.
6. [Understanding ROS Services - Clients](06-understanding-ros-services-clients.md) — What a ROS service is, how to manage services on a robot, and how to call one.
7. [Understanding ROS Services - Server](07-understanding-ros-services-server.md) — How to create a service and your own service message.
8. [Using C++ Classes in ROS](08-using-cpp-classes-in-ros.md) — How to create ROS programs using C++ classes.
9. [Understanding ROS Actions - Clients](09-understanding-ros-actions-clients.md) — What a ROS action is, how to manage actions on a robot, and how to call an action server.
10. [Understanding ROS Actions - Servers](10-understanding-ros-actions-servers.md) — How to create an action server and your own action message.
11. [How to Debug ROS Programs](11-how-to-debug-ros-programs.md) — Managing and filtering ROS logs, recording/replaying data, plotting topic data, visualizing the node graph, and basic RViz usage.
12. [Appendix](12-appendix.md) — Installing ROS locally and creating/managing ROS workspaces.
