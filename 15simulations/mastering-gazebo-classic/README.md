# Mastering Gazebo Classic

Gazebo Classic is the long-standing, physics-accurate simulator that a huge share of existing ROS robots and tutorials still target, and this course builds up the full workflow around it: drive the GUI and understand its process model, describe a mobile robot in URDF and get it standing in a world, wire that robot into the ROS graph with sensor and actuator plugins, author your own custom worlds and terrain, and finally drop down to C++ and write Gazebo plugins from scratch. It closes with a final project that combines all of it — a custom world, a hand-built four-wheeled robot, a dynamic obstacle driven by your own plugin, and an obstacle-avoidance controller you improve yourself.

1. [Gazebo Introduction](01-gazebo-introduction.md) — Learn how to work with the Gazebo GUI and how to launch it connected to a ROS environment.
2. [Build a Robot](02-build-a-robot.md) — Learn how to build a robot for Gazebo simulations using URDF.
3. [Connect to ROS](03-connect-to-ros.md) — Learn how to connect a Gazebo robot to ROS.
4. [Build Gazebo Worlds](04-build-gazebo-worlds.md) — Learn how to create a custom Gazebo world and models.
5. [Writing Gazebo Plugins](05-writing-gazebo-plugins.md) — Learn how to create Gazebo plugins.
6. [Final Project](06-final-project.md) — Combine everything into a robot that navigates a custom world while avoiding dynamic obstacles.
