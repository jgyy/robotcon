# Basic Maths for Robotics

Robots move, sense, and reason in a world that is fundamentally geometric, dynamic, and uncertain — which means the mathematics of geometry, change, and probability isn't optional background, it's the vocabulary the rest of robotics is written in. This course builds that vocabulary from the ground up: linear algebra for representing positions, orientations, and transformations; calculus for relating motion quantities and finding optimal parameters; and probability for reasoning about noisy sensors and uncertain state estimates. It closes with a small project that puts all three pillars to work driving a simulated robot out of a maze.

1. [Course Introduction](01-course-introduction.md) — why maths matters for robotics, what tools you'll use, and what to have set up before Unit 2.
2. [Linear Algebra (vectors and matrices)](02-linear-algebra-vectors-and-matrices.md) — vectors, matrices, linear maps, and the identities used constantly in robot geometry.
3. [Calculus](03-calculus.md) — functions, derivatives, partial derivatives, gradients, Jacobians, Hessians, and integrals.
4. [Probability](04-probability.md) — random variables, joint/conditional probability, Bayes' rule, common distributions, and beliefs.
5. [Final Project: Escape the maze!](05-final-project-escape-the-maze.md) — combine vectors, gradients, and Bayesian belief updates to navigate a simulated robot out of a maze.
